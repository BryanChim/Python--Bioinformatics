Parser_fastq.py accepts a FASTQ file of reads as input -- EXAMPLE: project.FASTQ -- then calculates and writes to output files:

Summary Statistics (read count, average/min/max read length and quality etc.) -- EXAMPLE: summary.txt

Average Quality at Each Position -- EXAMPLE: position_quality.txt

Nucleotide Counts and Frequencies at each Position -- EXAMPLE: position_acgt.txt
